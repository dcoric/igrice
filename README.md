# One Stop Shop Starter Kit

## Install from source

Clone the project:

```bash
$ git clone <git-url> <project name>
$ cd <project name>
```

Install dependencies by runing

```bash
$ yarn install    # Install project dependencies
$ yarn start      # Compile and launch (same as `npm start`)
```

or if you use npm

```bash
$ npm install
$ npm start
```
